# Public Folder

## English

**Purpose**

This folder stores static public assets served directly by Vite without going through the React bundle.

**What goes here**

- Favicons
- Static icons
- Images that should keep a fixed public URL
- Files referenced directly from HTML or metadata

**Notes**

Use this folder for assets that must be available at a predictable path.
If an asset is imported by React components, consider placing it under `src/assets` instead.

## Tiếng Việt

**Mục đích**

Thư mục này chứa các tài nguyên tĩnh được Vite phục vụ trực tiếp, không đi qua bundle React.

**Nên đặt gì ở đây**

- Favicon
- Icon tĩnh
- Ảnh cần giữ URL public cố định
- Các file được tham chiếu trực tiếp từ HTML hoặc metadata

**Ghi chú**

Hãy dùng thư mục này cho những asset cần truy cập bằng đường dẫn cố định.
Nếu asset được import từ component React, nên cân nhắc đặt trong `src/assets`.
