# Images

## English

**Purpose**

This folder stores image assets used by the application.

**What goes here**

- Icons
- Logos
- Illustrations
- Static image files used by components or pages

**Notes**

Use clear file names and group files by feature if the folder grows.
Prefer optimized image formats when possible.

## Tiếng Việt

**Mục đích**

Thư mục này dùng để chứa các ảnh tĩnh được sử dụng trong ứng dụng.

**Nên đặt gì ở đây**

- Icon
- Logo
- Illustration
- Các file ảnh tĩnh dùng trong component hoặc page

**Ghi chú**

Nên đặt tên file rõ ràng và nhóm theo feature nếu số lượng ảnh tăng lên.
Ưu tiên định dạng ảnh đã được tối ưu khi phù hợp.
