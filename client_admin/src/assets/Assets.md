# Assets

## English

**Purpose**

This folder stores source-controlled static assets that are imported by the application code.

**What goes here**

- Images
- Icons
- Illustrations
- Other static media bundled with the app

**Notes**

Group assets by type or feature when the folder grows.
Use `public` only for assets that must be served by a fixed URL.

## Tiếng Việt

**Mục đích**

Thư mục này chứa các tài nguyên tĩnh được quản lý trong source và được import bởi mã ứng dụng.

**Nên đặt gì ở đây**

- Ảnh
- Icon
- Illustration
- Các loại media tĩnh khác được bundle cùng app

**Ghi chú**

Khi số lượng asset tăng lên, nên nhóm theo loại hoặc theo feature.
Chỉ dùng `public` cho asset cần được phục vụ bằng URL cố định.
