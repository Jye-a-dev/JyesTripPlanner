# Providers

## English

**Purpose**

This folder contains React providers that wrap the application tree.

**What goes here**

- Context providers
- Theme providers
- Query or state providers
- Composition of shared app-level wrappers

**Notes**

Keep provider setup centralized here.
The root app component should stay thin and delegate provider composition to this folder.

## Tiếng Việt

**Mục đích**

Thư mục này chứa các React provider bọc quanh cây ứng dụng.

**Nên đặt gì ở đây**

- Context provider
- Theme provider
- Query hoặc state provider
- Phần ghép các wrapper dùng chung ở cấp ứng dụng

**Ghi chú**

Nên tập trung phần setup provider tại đây.
Component app gốc nên mỏng và giao phần ghép provider cho thư mục này.
