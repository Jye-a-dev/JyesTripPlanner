# Components

## English

**Purpose**

This folder contains reusable UI components and presentation building blocks.

**What goes here**

- Shared layout parts
- Reusable page sections
- Presentational components
- UI building blocks extracted from pages

**Notes**

Keep components reusable and focused.
Route ownership should stay in `src/app`, while shared UI stays here.

## Tiếng Việt

**Mục đích**

Thư mục này chứa các component UI tái sử dụng và các khối giao diện dùng để ghép trang.

**Nên đặt gì ở đây**

- Thành phần layout dùng chung
- Các section trang có thể tái sử dụng
- Component thiên về hiển thị
- UI được tách ra từ page để dùng lại

**Ghi chú**

Nên giữ component có trách nhiệm rõ ràng và dễ tái sử dụng.
Phần ownership theo route nên nằm ở `src/app`, còn UI dùng chung nên đặt ở đây.
