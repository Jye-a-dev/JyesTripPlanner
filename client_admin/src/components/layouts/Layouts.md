# Layout Components

## English

**Purpose**

This folder contains layout-specific components that define the page shell structure.

**What goes here**

- Public layout wrappers
- Base navbar and footer parts
- Layout-specific containers
- Shared structural UI for route shells

**Notes**

Separate layout composition from page content.
Use `@base` for foundational parts and route-specific folders for composition.

## Tiếng Việt

**Mục đích**

Thư mục này chứa các component liên quan đến layout, dùng để định nghĩa khung tổng thể của trang.

**Nên đặt gì ở đây**

- Wrapper layout public
- Các phần base như navbar và footer
- Container dành riêng cho layout
- UI cấu trúc dùng chung cho route shell

**Ghi chú**

Nên tách phần ghép layout ra khỏi nội dung page.
Dùng `@base` cho phần nền tảng và dùng các thư mục theo route để ghép layout cụ thể.
