# Base Layout Parts

## English

**Purpose**

This folder contains foundational layout primitives that can be reused by multiple layout variants.

**What goes here**

- Base navbar
- Base footer
- Shared structural layout components

**Notes**

Keep these components generic and reusable.
Route-specific behavior should stay outside this folder when possible.

## Tiếng Việt

**Mục đích**

Thư mục này chứa các thành phần layout nền tảng có thể được tái sử dụng bởi nhiều biến thể layout.

**Nên đặt gì ở đây**

- Base navbar
- Base footer
- Các component cấu trúc dùng chung

**Ghi chú**

Nên giữ các component này tổng quát và dễ tái sử dụng.
Logic riêng theo route nên được đặt ngoài thư mục này nếu có thể.
