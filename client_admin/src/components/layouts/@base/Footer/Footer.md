# Base Footer

## English

**Purpose**

This folder contains the foundational footer component shared across layouts.

**What goes here**

- Core footer markup
- Shared footer styling decisions
- Footer content that can be reused by multiple layout layers

**Notes**

Keep this component generic.
Public or feature-specific changes should usually be wrapped outside the base layer.

## Tiếng Việt

**Mục đích**

Thư mục này chứa component footer nền tảng được chia sẻ giữa nhiều layout.

**Nên đặt gì ở đây**

- Markup footer cốt lõi
- Quyết định style footer dùng chung
- Nội dung footer có thể tái sử dụng ở nhiều tầng layout

**Ghi chú**

Nên giữ component này ở mức tổng quát.
Các thay đổi riêng cho public hoặc feature thường nên được bọc ở ngoài base layer.
