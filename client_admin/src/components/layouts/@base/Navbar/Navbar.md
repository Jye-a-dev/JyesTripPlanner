# Base Navbar

## English

**Purpose**

This folder contains the foundational navbar component shared across layouts.

**What goes here**

- Core navbar markup
- Shared branding area
- Shared navigation structure

**Notes**

Keep the base navbar reusable.
Add route-specific or audience-specific logic in outer layout wrappers when needed.

## Tiếng Việt

**Mục đích**

Thư mục này chứa component navbar nền tảng được chia sẻ giữa nhiều layout.

**Nên đặt gì ở đây**

- Markup navbar cốt lõi
- Khu vực branding dùng chung
- Cấu trúc điều hướng dùng chung

**Ghi chú**

Nên giữ base navbar có khả năng tái sử dụng.
Nếu cần logic riêng theo route hoặc nhóm người dùng, hãy xử lý ở lớp wrapper bên ngoài.
