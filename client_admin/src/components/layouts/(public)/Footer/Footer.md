# Public Footer

## English

**Purpose**

This folder contains the footer component used by the public layout layer.

**What goes here**

- Public footer wrapper
- Public-only footer behavior or composition
- Footer pieces specific to public pages

**Notes**

Reuse base footer parts when possible.
Add logic here only if it is specific to the public-facing layout.

## Tiếng Việt

**Mục đích**

Thư mục này chứa component footer được dùng trong tầng layout public.

**Nên đặt gì ở đây**

- Wrapper footer cho public
- Hành vi hoặc phần ghép footer chỉ dành cho public
- Các phần footer đặc thù của trang public

**Ghi chú**

Nên tái sử dụng base footer khi có thể.
Chỉ thêm logic ở đây nếu nó thật sự dành riêng cho layout public.
