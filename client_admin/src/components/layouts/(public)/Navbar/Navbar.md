# Public Navbar

## English

**Purpose**

This folder contains the navbar component used by the public layout layer.

**What goes here**

- Public navbar wrapper
- Public navigation links
- Public-only navbar behavior

**Notes**

Keep shared visual structure in the base navbar when possible.
Use this folder for public-specific composition or route-facing navigation logic.

## Tiếng Việt

**Mục đích**

Thư mục này chứa component navbar được dùng trong tầng layout public.

**Nên đặt gì ở đây**

- Wrapper navbar cho public
- Link điều hướng public
- Hành vi navbar chỉ dành cho public

**Ghi chú**

Nên giữ phần cấu trúc dùng chung ở base navbar khi có thể.
Hãy dùng thư mục này cho phần ghép riêng của public hoặc logic điều hướng ở cấp route.
