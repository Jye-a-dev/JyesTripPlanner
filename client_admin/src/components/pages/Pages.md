# Page Components

## English

**Purpose**

This folder contains higher-level UI components that are specific to a page but still separated from the route file.

**What goes here**

- Page sections
- Page assemblies
- Reusable page-level blocks

**Notes**

Use this folder when a route page becomes too large and needs extracted UI.
Keep route entry files in `src/app`.

## Tiếng Việt

**Mục đích**

Thư mục này chứa các component UI cấp cao dành riêng cho page nhưng được tách khỏi file route.

**Nên đặt gì ở đây**

- Các section của page
- Phần ghép UI cho page
- Khối giao diện cấp page có thể tái sử dụng

**Ghi chú**

Nên dùng thư mục này khi route page trở nên quá lớn và cần tách UI ra.
File entry theo route vẫn nên nằm trong `src/app`.
