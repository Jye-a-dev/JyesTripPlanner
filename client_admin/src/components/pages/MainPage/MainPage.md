# Main Page Components

## English

**Purpose**

This folder contains UI parts used by the main public page.

**What goes here**

- Main page content blocks
- Hero or summary sections
- Reusable pieces extracted from the home page

**Notes**

Keep this folder focused on the main page domain.
Move truly shared UI to a more generic component folder if reuse expands.

## Tiếng Việt

**Mục đích**

Thư mục này chứa các phần UI được dùng bởi trang chính public.

**Nên đặt gì ở đây**

- Các khối nội dung của main page
- Section hero hoặc phần tóm tắt
- Các phần được tách ra từ home page để dễ quản lý

**Ghi chú**

Nên giữ thư mục này tập trung vào domain của main page.
Nếu mức độ tái sử dụng tăng lên, hãy chuyển UI thật sự dùng chung sang thư mục component tổng quát hơn.
