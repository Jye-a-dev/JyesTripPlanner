# Hooks

## English

**Purpose**

This folder is for custom React hooks shared across the application.

**What goes here**

- Data-fetching hooks
- UI state hooks
- Form or input helpers
- Reusable business logic extracted from components

**Notes**

Keep hooks focused and reusable.
If a hook is only used by one feature, consider placing it close to that feature first.

## Tiếng Việt

**Mục đích**

Thư mục này dùng để chứa các custom hook React được dùng lại trong toàn bộ ứng dụng.

**Nên đặt gì ở đây**

- Hook gọi dữ liệu
- Hook quản lý UI state
- Hook hỗ trợ form hoặc input
- Logic nghiệp vụ có thể tách khỏi component để tái sử dụng

**Ghi chú**

Nên giữ hook gọn, rõ trách nhiệm và có thể tái sử dụng.
Nếu hook chỉ phục vụ một tính năng duy nhất, có thể đặt gần feature đó trước.
