# Router

## English

**Purpose**

This folder contains route definitions and navigation structure for the application.

**What goes here**

- Route arrays
- Route objects
- Navigation metadata
- Route guard helpers if needed

**Notes**

Keep route declarations clear and close to layout ownership.
Avoid putting page implementation logic directly inside route config files.

## Tiếng Việt

**Mục đích**

Thư mục này chứa định nghĩa route và cấu trúc điều hướng của ứng dụng.

**Nên đặt gì ở đây**

- Mảng route
- Đối tượng route
- Metadata cho điều hướng
- Helper guard cho route nếu cần

**Ghi chú**

Nên giữ phần khai báo route rõ ràng và gần với ownership của layout.
Tránh đặt logic triển khai page trực tiếp trong file cấu hình route.
