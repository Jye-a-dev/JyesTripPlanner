# Source Folder

## English

**Purpose**

This is the main source folder of the client application.

**Structure overview**

- `app`: app-level screens and application composition
- `components`: reusable UI pieces and layouts
- `providers`: shared React providers
- `router`: route configuration
- `assets`: source-managed static assets
- `hooks`: reusable custom hooks
- `services`: API and side-effect modules
- `utils`: shared utility functions

**Notes**

Keep feature logic organized close to where it is used.
Use subfolder README files to understand the role of each area.

## Tiếng Việt

**Mục đích**

Đây là thư mục mã nguồn chính của ứng dụng client.

**Tổng quan cấu trúc**

- `app`: màn hình cấp ứng dụng và phần ghép ứng dụng
- `components`: UI dùng lại và layout
- `providers`: các React provider dùng chung
- `router`: cấu hình route
- `assets`: tài nguyên tĩnh được quản lý trong source
- `hooks`: custom hook tái sử dụng
- `services`: module API và side effect
- `utils`: hàm tiện ích dùng chung

**Ghi chú**

Nên tổ chức logic theo vị trí sử dụng thực tế.
Có thể xem thêm các file README trong từng thư mục con để hiểu vai trò cụ thể.
