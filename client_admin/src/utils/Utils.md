# Utils

## English

**Purpose**

This folder contains small reusable utility functions that do not belong to React components, hooks, or services.

**What goes here**

- Formatters
- Parsers
- Validation helpers
- Common pure functions

**Notes**

Utilities should stay framework-light and easy to test.
Avoid turning this folder into a dumping ground for unrelated logic.

## Tiếng Việt

**Mục đích**

Thư mục này chứa các hàm tiện ích nhỏ có thể tái sử dụng, không thuộc về component, hook hay service.

**Nên đặt gì ở đây**

- Hàm format
- Hàm parse
- Helper validation
- Các pure function dùng chung

**Ghi chú**

Utils nên nhẹ, dễ test và ít phụ thuộc framework.
Tránh biến thư mục này thành nơi gom mọi logic không rõ chỗ đặt.
