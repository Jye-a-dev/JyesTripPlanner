# Home Page

## English

**Purpose**

This folder contains the public home page entry for the application.

**What goes here**

- Home page container
- Home-specific sections if the page grows
- Page-level composition for the landing screen

**Notes**

Keep reusable sections in `src/components` when they start being shared.
This folder should stay focused on the route-level home page assembly.

## Tiếng Việt

**Mục đích**

Thư mục này chứa page trang chủ public của ứng dụng.

**Nên đặt gì ở đây**

- Container của trang chủ
- Các section riêng của home nếu trang lớn dần
- Phần ghép page cho màn hình landing

**Ghi chú**

Nếu section bắt đầu được dùng lại, nên chuyển sang `src/components`.
Thư mục này nên tập trung vào việc ghép page ở cấp route.
