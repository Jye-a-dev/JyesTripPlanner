# Public App Pages

## English

**Purpose**

This folder contains route pages that are accessible without authentication.

**What goes here**

- Landing pages
- Marketing pages
- Public home page
- Public informational screens

**Notes**

Pages in this folder usually render inside the public layout.
Keep public route pages separate from authenticated user pages.

## Tiếng Việt

**Mục đích**

Thư mục này chứa các page có thể truy cập mà không cần đăng nhập.

**Nên đặt gì ở đây**

- Landing page
- Trang marketing
- Trang chủ public
- Các màn hình thông tin công khai

**Ghi chú**

Các page trong thư mục này thường được render bên trong public layout.
Nên tách riêng page public khỏi page dành cho người dùng đã đăng nhập.
