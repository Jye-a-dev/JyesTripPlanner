# Not Found Page

## English

**Purpose**

This folder contains the fallback page shown when no route matches.

**What goes here**

- 404 page component
- Missing-route helpers
- Empty-state or recovery links for unmatched routes

**Notes**

Keep this page simple and clear.
It should help users return to a valid route quickly.

## Tiếng Việt

**Mục đích**

Thư mục này chứa trang fallback được hiển thị khi không có route nào khớp.

**Nên đặt gì ở đây**

- Component trang 404
- Helper cho route không tồn tại
- Liên kết quay về trang hợp lệ khi người dùng vào sai đường dẫn

**Ghi chú**

Nên giữ trang này đơn giản và rõ ràng.
Mục tiêu là giúp người dùng quay lại route hợp lệ nhanh chóng.
