# App Folder

## English

**Purpose**

This folder contains application-level entry screens and route-oriented page modules.

**What goes here**

- Root application component
- Route page components
- App-level CSS if still needed
- Reserved folders for public, user, auth, and fallback screens

**Notes**

Use this folder for route-facing page modules, not low-level reusable UI.
Shared UI should usually live in `src/components`.

## Tiếng Việt

**Mục đích**

Thư mục này chứa các màn hình cấp ứng dụng và các module page gắn với route.

**Nên đặt gì ở đây**

- Component gốc của ứng dụng
- Component page theo route
- CSS cấp ứng dụng nếu còn cần
- Các thư mục dành riêng cho public, user, auth và trang fallback

**Ghi chú**

Nên dùng thư mục này cho module page hướng route, không phải UI tái sử dụng mức thấp.
UI dùng chung nên đặt ở `src/components`.
