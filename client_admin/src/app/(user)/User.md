# User App Pages

## English

**Purpose**

This folder is reserved for pages that require an authenticated user context.

**What goes here**

- User dashboard pages
- Profile pages
- Account pages
- Private route screens

**Notes**

Use this area for authenticated flows only.
If access control becomes more complex, keep guard logic near the router or layout layer.

## Tiếng Việt

**Mục đích**

Thư mục này dành cho các page cần ngữ cảnh người dùng đã đăng nhập.

**Nên đặt gì ở đây**

- Trang dashboard người dùng
- Trang hồ sơ
- Trang tài khoản
- Các màn hình private route

**Ghi chú**

Chỉ nên dùng khu vực này cho luồng đã xác thực.
Nếu logic phân quyền phức tạp hơn, nên đặt guard gần router hoặc layer layout.
